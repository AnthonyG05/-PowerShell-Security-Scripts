Project Overview

A specialized toolkit of PowerShell scripts designed for administrative automation, network monitoring, and cryptographic operations. These utilities were developed to support "Blue Team" defensive strategies and incident response workflows.
Key Utilities
    Network Connection Monitor: Audits active TCP/IP connections, mapping remote IP addresses to local PIDs and process names for rapid threat identification.
    File Integrity Checker: Utilizes SHA256 hashing algorithms to generate and verify file signatures, ensuring data remains untampered across local directories.
    Cryptographic Operations: Features functions for Base64 encoding/decoding and SecureString encryption to demonstrate safe data-at-rest practices.
    Low-Level Memory Management: Employs the Runtime.InteropServices.Marshal class for advanced decryption tasks, bypassing standard high-level limitations.
    

Technical Stack
    Language: PowerShell
    Focus: Hashing (SHA256), Cryptography, Network Auditing
